# [](/plugin/macrosamples/)Insert text

Description

Inserts text into the document at the current cursor position.

```
(function()
{
    var oDocument = Api.GetDocument();
    var oParagraph = Api.CreateParagraph();
    oParagraph.AddText("Hello world!");
    oDocument.InsertContent([oParagraph]);
})();
```

Methods used: [GetDocument](/docbuilder/textdocumentapi/api/getdocument), [CreateParagraph](/docbuilder/textdocumentapi/api/createparagraph), [AddText](/docbuilder/textdocumentapi/apiparagraph/addtext), [InsertContent](/docbuilder/textdocumentapi/apidocument/insertcontent)
