# [](/plugin/macrosamples/)Write data to worksheet cell

Description

Writes the data (the *Hello world* phrase) to the worksheet third column of the fourth row.

```
(function()
{
    Api.GetActiveSheet().GetRange("C4").SetValue("Hello world");
})();
```

Methods used: [GetActiveSheet](/docbuilder/spreadsheetapi/api/getactivesheet), [GetRange](/docbuilder/spreadsheetapi/apiworksheet/getrange), [SetValue](/docbuilder/spreadsheetapi/apirange/setvalue)

Reference Microsoft VBA macro code

```
Sub example()
    Cells(3, 4)="Hello world"
End Sub
```
