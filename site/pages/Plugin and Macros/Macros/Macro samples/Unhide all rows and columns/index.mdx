# [](/plugin/macrosamples/)Unhide all rows and columns

Description

Unhides all the rows and columns in the active spreadsheet.

```
(function()
{
    var activeSheet = Api.ActiveSheet;
    var indexRowMax = 1048576;
    var n = 1;
    for (let i = 0; i < indexRowMax; i++) {
        activeSheet.GetRows(n).SetHidden(false);
        n++;
    }
    var newRange = activeSheet.GetRange("A1");
    newRange.SetValue("All the rows and columns are unhidden now");
})();
```

Methods used: [GetActiveSheet](/docbuilder/spreadsheetapi/api/getactivesheet), [GetRows](/docbuilder/spreadsheetapi/apirange/getrows), [SetHidden](/docbuilder/spreadsheetapi/apirange/sethidden), [GetRange](/docbuilder/spreadsheetapi/api/getrange), [SetValue](/docbuilder/spreadsheetapi/apirange/setvalue)
