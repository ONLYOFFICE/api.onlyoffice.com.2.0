# [](/plugin/macrosamples/)Recalculate worksheet values

Description

Repeatedly recalculates worksheet cell values within one-second intervals.

```
(function ()
{
    let timerId = setInterval(function(){
        Api.asc_calculate(Asc.c_oAscCalculateType.All);
        var oWorksheet = Api.GetActiveSheet();
        oWorksheet.GetRange("F2").SetValue("Average");
        oWorksheet.GetRange("G2").SetValue("=AVERAGE(B2:B80)");
    }, 1000);
})();
```

Methods used: [GetActiveSheet](/docbuilder/spreadsheetapi/api/getactivesheet), [GetRange](/docbuilder/spreadsheetapi/api/getrange), [SetValue](/docbuilder/spreadsheetapi/apirange/setvalue)
