# [](/plugin/macrosamples/)Change cell background color

Description

Sets the background color of the cell **B3** to blue.

```
(function()
{
    Api.GetActiveSheet().GetRange("B3").SetFillColor(Api.CreateColorFromRGB(0, 0, 250));
})();
```

Methods used: [GetActiveSheet](/docbuilder/spreadsheetapi/api/getactivesheet), [GetRange](/docbuilder/spreadsheetapi/apiworksheet/getrange), [SetFillColor](/docbuilder/spreadsheetapi/apirange/setfillcolor), [CreateColorFromRGB](/docbuilder/spreadsheetapi/api/createcolorfromrgb)

Reference Microsoft VBA macro code

```
Sub example()
    Range("B3").Interior.Color = RGB(0, 0, 250)
End Sub
```
