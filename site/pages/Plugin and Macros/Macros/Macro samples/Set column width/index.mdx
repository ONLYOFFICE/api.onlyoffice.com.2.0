# [](/plugin/macrosamples/)Set column width

Description

Sets the width for the column **B**.

```
(function()
{
    Api.GetActiveSheet().SetColumnWidth(1, 25);
})();
```

Methods used: [GetActiveSheet](/docbuilder/spreadsheetapi/api/getactivesheet), [SetColumnWidth](/docbuilder/spreadsheetapi/apiworksheet/setcolumnwidth)

Reference Microsoft VBA macro code

```
Sub example()
    Columns("B").ColumnWidth = 25
End Sub
```
