# [](/plugin/macrosamples/)Merge cell range

Description

Merges the selected cell range.

```
(function()
{
    Api.GetActiveSheet().GetRange("A1:B3").Merge(true);
})();
```

Methods used: [GetActiveSheet](/docbuilder/spreadsheetapi/api/getactivesheet), [GetRange](/docbuilder/spreadsheetapi/apiworksheet/getrange), [Merge](/docbuilder/spreadsheetapi/apirange/merge)

Reference Microsoft VBA macro code

```
Sub example()
    Range("A1:B3").Merge
End Sub
```
