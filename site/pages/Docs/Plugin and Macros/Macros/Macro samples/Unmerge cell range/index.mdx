# [](/plugin/macrosamples/)Unmerge cell range

Description

Unmerges the selected cell range.

```
(function()
{
    Api.GetActiveSheet().GetRange("C5:D10").UnMerge();
})();
```

Methods used: [GetActiveSheet](/docbuilder/spreadsheetapi/api/getactivesheet), [GetRange](/docbuilder/spreadsheetapi/apiworksheet/getrange), [UnMerge](/docbuilder/spreadsheetapi/apirange/unmerge)

Reference Microsoft VBA macro code

```
Sub example()
    Range("C5:D10").UnMerge
End Sub
```
