# [](/plugin/macrosamples/)Change cell font color

Description

Sets the font color of the cell **B4** to red.

```
(function()
{
    Api.GetActiveSheet().GetRange("B4").SetFontColor(Api.CreateColorFromRGB(255, 0, 0));
})();
```

Methods used: [GetActiveSheet](/docbuilder/spreadsheetapi/api/getactivesheet), [GetRange](/docbuilder/spreadsheetapi/apiworksheet/getrange), [SetFontColor](/docbuilder/spreadsheetapi/apirange/setfontcolor)

Reference Microsoft VBA macro code

```
Sub example()
    Range("B4").Font.Color = RGB(255, 0, 0)
End Sub
```
