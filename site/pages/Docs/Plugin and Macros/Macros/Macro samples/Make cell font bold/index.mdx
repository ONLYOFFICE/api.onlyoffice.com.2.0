# [](/plugin/macrosamples/)Make cell font bold

Description

Sets the font of the cell **A2** to bold.

```
(function()
{
    Api.GetActiveSheet().GetRange("A2").SetBold(true);
})();
```

Methods used: [GetActiveSheet](/docbuilder/spreadsheetapi/api/getactivesheet), [GetRange](/docbuilder/spreadsheetapi/apiworksheet/getrange), [SetBold](/docbuilder/spreadsheetapi/apirange/setbold)

Reference Microsoft VBA macro code

```
Sub example()
    Range("A2").Font.Bold = True
End Sub
```
